# Platform Content Governance UI

## Purpose

The Platform Content Governance UI gives Vytte Platform Admins a single control center for official content without creating a second assessment system.

## UI areas

- Official Content dashboard: counts and entry points for governed objects.
- Question Groups: group reusable questions inside official departments/focused scopes.
- Question Identities: create stable reusable questions and first draft versions.
- Question Versions: approve and publish immutable question content.
- Framework Versions: inspect sections, indicators, placements, source metadata, scoring version, and publish valid drafts.
- Catalogue Releases: inspect pinned framework manifests and publish valid releases.
- Facility Profiles: inspect which departments/services are valid for each facility profile.
- Scoring Policies: inspect scoring profile versions, aggregation policies, and multi-respondent collection settings.
- Analytical Domains: inspect domain taxonomies and mappings.

## Publication principle

The UI calls existing governed services for publication. It does not directly mark official content as published without validation.

## Naming principle

The product uses **Question Group** for structural question grouping and **Vytte Platform Admin** for platform governance authority.
